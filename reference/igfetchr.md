# igfetchr: Lightweight IG Trading REST API wrapper

Helper functions to authenticate and fetch market/account data from the
IG Trading REST API. All exported functions return tibbles where
appropriate. This package is not financial advice. Trading CFDs and
spread bets carries high risk of losing money. Key functions:
\`ig_auth\` to authenticate, \`ig_execute_trade\` to place trades.
